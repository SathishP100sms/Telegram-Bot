# 🤖 AI Telegram Bot (FastAPI + Gemini)

A powerful AI-powered Telegram chatbot built using **FastAPI**, **python-telegram-bot**, and **Google Gemini API**.

---

## 🚀 Features

- 💬 Smart AI chat responses
- ⚡ FastAPI webhook server
- 🤖 Telegram bot integration
- 🧠 Context-aware responses
- 🔒 Secure environment variable usage
- ☁️ Easy deployment on Render

---

## 🛠️ Tech Stack

- **Backend**: FastAPI  
- **Bot Framework**: python-telegram-bot  
- **AI Model**: Google Gemini  
- **Server**: Uvicorn
